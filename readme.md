# Como entrar no chat
## Requisitos
    * Node

## Iniciando o servidor
Abra um terminal na pasta do projeto e escreva o comando abaixo e então aperte ENTER
```
node .\index.ts
```
## iniciando o client
Abra um terminal na pasta do projeto e escreva o comando abaixo e então aperte ENTER
```
node .\client.ts
```
